## Usage

-   With the HP Visual Identity System, you can choose from over 300 brand icons.
-   Make sure you only use the icons designated by HP (see link below). Do not create or use any other icons.
-   Do not use icons in other modules aside from the Icon Link and Icon List modules.

## Guidelines

-   Choose icons that correlate with the benefit or feature that you’re highlighting.
-   Icons are formatted with matching stroke widths. Please do not resize icons.
-   Icon’s color matches the Signifier’s color.
