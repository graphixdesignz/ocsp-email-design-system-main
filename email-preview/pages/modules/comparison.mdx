# Comparison

Similar to the Dual Image module, the Comparison module places two columns with images side by side.
This module differs from Dual Image in that it supports more detailed content with the intention
of drawing comparisons between products or services.

## Guidelines

-   Images in each column should have the same dimensions.
-   Recommend images with a 4:3 aspect ratio
