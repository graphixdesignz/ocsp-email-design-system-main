# Dual Image

Two horizontal columns featuring an image, subhead, copy, and a soft CTA.
Useful for linking to content of equal priority, e.g. two articles on a common topic.

## Guidelines

-   Images in each column should have the same dimensions.
-   Recommend images with a 4:3 aspect ratio
