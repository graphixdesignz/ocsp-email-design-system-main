# Responsive Image

A full bleed responsive image. Image will fill the container width at 100% (max 600px on desktop) and scale _down_
proportionally

## Guidelines

-   Most images should be proportional to 1200px x 800px to display at 600px x 400px on high resolution displays.
-   Images may be formatted as banners; these should be proportional to 1200px x 320px to display at 600px x 160px on high resolution displays.
-   Images should be no taller than 450px (or 900px for high resolution displays).
-   Be sure to add a description in the images "alt" attribute when appropriate.
