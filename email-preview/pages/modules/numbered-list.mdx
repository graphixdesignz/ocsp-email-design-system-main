# Numbered List

An ordered, numbered list.

### Usage Note

Numbers must be manually updated. List is created using an HTML Table instead of traditional ordered-list `<OL>` element.

Ordered Lists are not broadly compatible with email clients, therefore a table is used to ensure
compatibility. This does require manually editing numbers in the list.

-   Recommend a single line of copy per list item
