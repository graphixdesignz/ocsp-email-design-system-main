## How to use recipes

Begin by copying and pasting the master layout module into your email authoring environment
to create the base template. The master layout provides the responsive layout and global typographic styles.

Then select and use a recipe:

-   Select a colorway from the available options.
-   Navigate to the recipe that best fits your needs. The recipe should preview in the selected color.
-   If needed, toggle the selector for Right to Left language.
-   Scroll to the provided reference HTML. This HTML markup has been automatically updated to include your selections.
-   Copy the HTML and paste it into your email authoring environment.
